Mengenal Semantic HTML

Oleh: Caesar

Semantic HTML adalah penggunaan tag HTML yang sesuai dengan makna kontennya, bukan hanya sekadar tampilan.

Contoh tag semantic meliputi <header>, <footer>, dan <article>.

Manfaat Semantic Web

  • Memudahkan mesin pencari (SEO) memahami konten.
  • Meningkatkan aksesibilitas bagi pengguna dengan alat bantu baca layar.
  • Memudahkan pengembang dalam membaca struktur kode.
